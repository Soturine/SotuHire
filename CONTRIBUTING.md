# Contributing

Obrigado pelo interesse em contribuir com o SotuHire.

O SotuHire é um projeto de IA aplicada à busca e análise de vagas. A contribuição ideal melhora clareza, utilidade, qualidade de código ou documentação sem transformar o MVP em um sistema desnecessariamente complexo.

## Princípios

- Mantenha o MVP simples.
- Prefira código legível a arquitetura exagerada.
- Separe UI, regras de negócio e integração com IA.
- Escreva testes para lógica determinística.
- Não commite currículos reais, `.env`, chaves ou dados pessoais.
- Documente decisões importantes.

## Como contribuir

1. Crie uma branch com nome claro.
2. Faça uma alteração pequena e focada.
3. Rode testes.
4. Atualize docs se necessário.
5. Abra PR explicando o motivo da mudança.

## Checklist de PR

- [ ] O código roda localmente.
- [ ] Os testes passam.
- [ ] A mudança tem escopo claro.
- [ ] Não há dados pessoais.
- [ ] Não há chaves de API.
- [ ] A documentação foi atualizada quando necessário.
