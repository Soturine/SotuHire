# v1.9.5 — Product QA, Demo Polish & Portfolio Launch

Release de fechamento de gaps, não de reescrita. A auditoria percorreu core, API, web, extensão, scripts, testes, docs e workflows.

## Correções implementadas

- linhagem e deduplicação compartilhadas para Perfil, memória, oportunidade, projeto e edital;
- `source_ref` preservado até o Career Context e a UI de rastreabilidade;
- metadata de IA com provider/modelo solicitado e usado, prompt/versionamento, fallback e evidências;
- GitHub passa a receber contexto confirmado quando local ou autorizado;
- extensão v0.9.1 com UI moderna, fila offline, IA local/SotuHire/Gemini/OpenAI e segredo isolado;
- catálogo oficial de modelos e enriquecimento público de perfis/repositórios GitHub;
- sete personas coerentes ligando Perfil, evidência, oportunidade, wishlist, Tracker, notificação e contexto;
- Dashboard agregado a partir dos stores existentes;
- remoção de implementação antiga de Settings e correção de mocks desatualizados;
- contrato real de `pip install -e .[dev]` no `pyproject.toml`;
- scripts de clean install e captura completa.

## Segurança mantida

Sem auto-apply, inscrição, pagamento, envio de documentos, crawler logado novo, CAPTCHA bypass ou decisão crítica só por IA. A chave do app fica no backend; chaves próprias opcionais da extensão ficam no service worker, em sessão por padrão ou cofre IndexedDB após consentimento. `authenticated-browser` permaneceu intocado.

## Roadmap honesto

PDF/HTML direto de Lattes/edital, calendário de estudo avançado, notificações nativas e telemetria histórica de IA continuam fora desta release.
