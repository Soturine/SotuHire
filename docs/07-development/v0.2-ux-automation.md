# v0.2 — UX guiada e automação básica

## Entrega

A v0.2 substitui o formulário único por um fluxo guiado:

1. Currículo;
2. Vaga;
3. Preferências;
4. Resultado.

Histórico e Dashboard aparecem como abas adicionais.

## Modo rápido

O caminho principal é:

```text
subir currículo + colar vaga + revisar detecção + analisar
```

Preferências avançadas ficam recolhidas por padrão. O usuário não precisa preencher manualmente cargo, modalidade, contrato, salário e senioridade quando o parser consegue detectá-los.

## Modo avançado

Expande preferências de localização, modalidade, salário, contrato, senioridade e notas prioritárias.

## Revisão assistida

Os schemas `ResumeProfileSchema` e `JobPostingSchema` são exibidos para correção antes da análise. Extração automática é sugestão, não verdade final.

Parsers aceitam texto vazio, detectam campos simples e leem TXT/DOCX. PDF é suportado por PyMuPDF.
