# Checklist de instalação limpa — v1.9.5

## Ambiente

- Python 3.11+ e `python -m venv .venv`.
- Node 22 recomendado e npm compatível com o lockfile.
- Git disponível.
- Portas: API 8787, Companion 8765, web 5173; podem ser ajustadas localmente.

## Instalação

```bash
python -m venv .venv
# ativar a venv
python -m pip install --upgrade pip
pip install -e ".[dev]"
cd apps/web
npm ci
cd ../..
```

## Verificação automatizada

```bash
python scripts/verify_clean_install.py --skip-e2e
```

Use `--install` para criar outra venv dentro do checkout exportado e validar o contrato de dependências. Sem `--skip-e2e`, o script executa Chromium em porta isolada.

## Execução

```bash
python scripts/run_api.py
cd apps/web && npm run dev
```

Opcionalmente inicie a Local Companion conforme `browser-extension/README.md`. O site funciona sem extensão; a extensão funciona sem React quando a Companion está ativa.

## Critérios

- API `/api/v1/health` responde com v1.9.5.
- Demo abre sem provider.
- API Real usa apenas localhost/CORS permitido.
- `npm ci`, build, lint, typecheck e smoke E2E passam.
- ZIP v0.9.1 é gerado sem segredos.
- MkDocs estrito e links do README passam.
