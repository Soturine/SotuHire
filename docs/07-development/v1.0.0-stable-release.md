# v1.0.0 — Stable Multi-Domain Career Intelligence Platform

## Objetivo

A v1.0.0 consolida o SotuHire como versão estável e demonstrável de portfolio, sem transformar a
release em uma refatoração grande.

## Entregas principais

- Apresentação melhor do Match Engine 2.0 no fluxo principal.
- Integração entre Match Engine 2.0, ATS review e Resume Tailor.
- Pesos configuráveis por domínio no matching.
- Demos fictícias multiárea.
- GitHub Pages como site estático de produto/documentação.
- Documentação final de release.

## Escopo preservado

- O app completo continua local, via Streamlit.
- GitHub Pages permanece estático.
- Coleta autenticada não foi alterada.
- Local Companion API continua local.
- Regras anti-invenção continuam obrigatórias.
