# v1.9.2 — Lattes, Academic Profile & AI-Assisted Evidence Extraction

A v1.9.2 amplia o SotuHire para trajetórias acadêmicas, científicas, técnicas, artísticas, de extensão e docência, sem criar um perfil separado.

## Entregas

- `modules/academic` com parser local de texto Lattes e serviço de confirmação.
- Endpoints `/api/v1/profile/import-lattes`, `/api/v1/profile/lattes/draft` e `/api/v1/profile/lattes/confirm`.
- Prompt Registry com `profile_lattes_extractor_v1`.
- Tela `/profile` com seção **Acadêmico / Lattes**.
- Career Context Engine com purposes `academic`, `lattes` e `public_exams`.
- Evidências acadêmicas em Match, ATS, Tailor, GitHub/Portfólio e Tracker por meio do Perfil Profissional Universal.
- Fundação documental para editais e concursos, sem implementar Concurso Mode completo.

## Regra central

Nada extraído do Lattes entra automaticamente no Perfil. O fluxo é:

```text
texto colado -> parser local e/ou Gemini -> candidatos revisáveis -> confirmação explícita -> Perfil Profissional Universal
```

## IA opcional

Gemini pode organizar o texto e sugerir candidatos, mas não decide verdade factual. Se o provider falhar ou retornar JSON inválido, o SotuHire usa o parser local e emite warning.

## Segurança

- Sem login no Lattes.
- Sem scraping autenticado.
- Sem crawler.
- Sem candidatura automática.
- Sem inscrição automática em concursos ou editais.
- Sem API key do app no frontend.
- Sem alteração no fluxo authenticated-browser.
