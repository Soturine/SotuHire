# v0.3 — IA estruturada, Resume Tailor e exports

## Provider layer

A análise estruturada usa contrato único e Pydantic:

```text
UI -> analyze_structured -> provider -> JobAnalysisSchema
```

Providers:

- mock/local por padrão;
- Gemini opcional;
- fallback local em falha.

## Configuração

```text
DEFAULT_AI_PROVIDER=mock
GEMINI_API_KEY=
```

Para Gemini:

```bash
pip install -r requirements-ai.txt
```

## Resume Tailor

Entrega resumo profissional, bullets normalizados, ordem sugerida, keywords seguras, warnings e evidências. A saída rejeita `invented_information=True`.

## Exports

Downloads disponíveis:

- análise JSON;
- análise Markdown;
- Resume Tailor Markdown.

A pasta local `exports/` é ignorada pelo Git. DOCX/PDF final continua fora do escopo.

## Testes
