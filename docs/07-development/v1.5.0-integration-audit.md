# v1.5.0 — auditoria de integração

Data: 2026-06-23.

Objetivo: auditar o que existe no core, o que tem API, o que aparece no frontend moderno e onde a
IA opcional usa o Prompt Registry. Esta auditoria não altera os documentos protegidos de compliance
e não adiciona automação de login, auto-apply, crawler logado novo ou bypass de CAPTCHA.

## Observações

- O frontend continua sem calcular score real e sem persistir API key.
- A chave de IA fica em `data/secrets/ai-provider.local.json`, ignorado pelo Git.
- Gemini é usado apenas quando `use_ai=true`, provider configurado e toggle da área permitido.
- Falha de Gemini retorna fallback local com warning; a análise não quebra.
- A ponte `/api/v1/extension/*` consulta capturas locais já existentes da Local Companion API; ela
  não altera a extensão, Chromium/CDP, authenticated browser ou regras protegidas.
- Streamlit permanece disponível como legado/dev/local debug.
