# Regras futuras para Concurso Mode

Este documento existe para registrar o potencial do modo concurso sem misturar escopo com o MVP de vagas privadas.

## Regras de extração

O parser de edital deve identificar:

- órgão;
- banca;
- cargo;
- escolaridade;
- salário;
- benefícios;
- vagas imediatas;
- cadastro reserva;
- regime jurídico;
- taxa;
- datas;
- conteúdo programático;
- etapas;
- pesos.

## Regras de comparação

Comparar edital novo com antigo:

- banca mudou?
- salário mudou?
- vagas mudaram?
- conteúdo aumentou?
- conteúdo reduziu?
- prova discursiva entrou/saiu?
- regime mudou para CLT?

## Score de custo-benefício

O score futuro pode considerar:

- salário;
- estabilidade/regime;
- vagas;
- concorrência estimada;
- tempo até prova;
- volume de conteúdo;
- alinhamento com formação do usuário;
- distância/local de prova.

## Guardrail

O SotuHire não deve prometer aprovação. Ele só pode organizar informação e apoiar decisão.
