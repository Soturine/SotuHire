# Resumo ficticio de memoria de carreira

Este exemplo demonstra como o SotuHire pode consolidar fatos locais sem usar dados pessoais reais.

## Perfil

- Cargo alvo: Analista de Dados Junior
- Skills fortes: Python, SQL e Power BI
- Modalidades preferidas: remoto e hibrido
- Gap recorrente: dbt

## Evidencias recentes

- Painel de indicadores ficticio com Python, SQL e Power BI
- Analise de vaga com recomendacao `apply_with_adjustments`

## Privacidade

Por padrao, estes dados permanecem locais. O Gemini recebe somente um resumo relevante quando a pessoa habilita explicitamente essa opcao.
